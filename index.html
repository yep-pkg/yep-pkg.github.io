Simple Commands

Intuitive commands that just make sense

# Install packages
$ yep install express react
# Search for packages
$ yep search lodash
# Remove packages
$ yep remove unused-package
# Update all packages
$ yep update
# Show package info
$ yep info package-name
# List installed packages
$ yep list
